# Glyphs 實驗工具集

這裡堆放一些還不到結構簡單或太惡搞，不想放上 Glyphs 外掛管理員的外掛作品。
以及外掛以外的 Glyphs 相關自製資源（如字表、腳本等）。


各工具的說明請參考各資料夾內，可用 Git 工具 Clone 使用。
或是也可以下載此資料夾的同名 .zip 檔解壓使用。
